
Knight's Tour Solver - User Instructions

1. Chessboard Setup:

Enter the desired chessboard size in the "Enter Chessboard Size" field. The chessboard will be an N x N grid.
2. Knight's Starting Position:

Specify the starting position of the knight by entering the row and column in the respective fields.
Make sure the values are valid positive integers within the chessboard size.
3. Algorithm Selection:

Choose the solving algorithm by selecting one of the radio buttons:
Backtracking: Uses a depth-first search algorithm.
Genetic Algorithm: Applies evolutionary principles to find a solution.
4. Genetic Algorithm Parameters (if selected):

If you choose the Genetic Algorithm, provide additional parameters:
Population Size: The number of individuals in the genetic algorithm population.
Generations: The number of generations the genetic algorithm will run.
Mutation Rate: The probability of mutation for each gene in an individual.
5. Initiating the Solution:

Click the "Solve" button to start the solving process.
6. Observe the Solution:

The chessboard and the solving process will be displayed on the Canvas.
For the backtracking algorithm, the knight's moves will be shown gradually.
For the genetic algorithm, the best solution and its fitness will be printed in the console.
7. Results:

If a solution is found, the chessboard will display the knight's path.
If no solution is found, a message will inform you.
8. Warning for Large Boards:

If the chessboard size is greater than 64, a warning will be displayed as computing time may be lengthy.
9. Exit:

Close the application window to exit.
Note:

The application provides a visual representation of the Knight's Tour problem-solving process, making it an interactive and educational tool. Enjoy exploring different chessboard sizes and algorithms!
